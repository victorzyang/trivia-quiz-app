How to run program:
- Go to terminal and enter: node trivia-server.js
- Then open "http://localhost:3000"

Add-ons implemented:
- Chat functionality, with an option to send messages privately to another player.
- A timer that allows players only 30 seconds to answer a question. For every 10 seconds that pass, the amount a player can score decreases. For example, if the player answers a question correctly within 10 seconds they will score 100 points. If the player answers correctly within 10 to 20 seconds, the user's score will be 75 points. When there are 10 seconds remaining, an incorrect answer will be disabled.
